# Counteroffer Builder

Use this skill when the user asks:

- "What should I counter with?"
- "How do I get this done?"
- "What should I add?"
- "What should I ask for?"
- "What should I send back?"
- "How do I negotiate this?"

## Workflow

1. Identify both teams' needs.
2. Identify my leverage.
3. Identify their likely motivation.
4. Build one conservative counter.
5. Build one fair counter.
6. Build one ambitious counter.
7. Recommend which one to send first.
8. Give a walk-away line.
9. Include a short paste-ready message.
10. Save negotiation notes if the trade remains active and the user asks to remember it.

## Output Format

1. Current offer summary
2. My leverage
3. Their likely motivation
4. Conservative counter
5. Fair counter
6. Ambitious counter
7. Recommended first offer
8. Walk-away line
9. Paste-ready message

## Guardrails

- Keep counters realistic enough that the other manager has a reason to engage.
- Do not auto-send a counteroffer.
- Stop for explicit approval before any state-changing action.
